2010【前言】小弟於八月底到目前九月，大概有一個多月的時間都專心在找工作，至此已經面試14間公司﹝台達電、宏達電、鴻揚、系微、鼎新、和碩、奇鋐、行動智慧、Career就業情報、和泰汽車、愛歌、宏正自動、優必達、口袋移動﹞，有些是主動投履歷，有些是對方邀約面試，那接下來我就開始分享面試的心得，希望能幫到一些人。【背景簡介】●今年八月底退伍●私立資訊管理碩士畢業【宏達電 (HTC)、台達電 (DELTA)、鴻揚】這三間全部貢龜，宏達電是小弟於當兵大概五月時，想說早點卡位置就勇敢地給他投下去，沒想到人資隔天就打給我約面試時間，效率非常快，貢龜也很快，由於是第一間加上還在當兵，都沒啥準備，當天考了英文、C、C++、邏輯測驗，有兩科沒過，就沒面試機會。第二間台達電，也是先考筆試，內容蠻廣的，出的像問答題，小弟在寫的時候也是腦袋不靈光，考完才覺得很多題自己可以寫得更好，一樣沒通知複試，掛彩。第三間鴻揚，這間是做企業專業之軟體顧問服務與軟體委外服務，面試過程先考邏輯測驗，之後上機考，小弟上機考差一題就可面試，一樣失敗，後來看板上原來有鴻揚的一些資訊，悔恨....。【系微 (Insydesw)】●職位：BIOS engineer●待遇(萬/月)：N-4●工時(小時/日)：9~10 以上●面試流程：筆試→主管面試這間公司大部分是在做 BIOS 的開發，筆試的部分考了 C語言與組語、邏輯、英文，考完之後似乎還可以，順利地跟主管面試，原來那主管是看到我自傳的某一句話有打中他的心才找我面試的，要不然通常是不會找資管的來，工作內容還蠻硬的，主管說他有一次要趕 Case 曾經有兩天待在公司 coding，我想若在這間公司練完功應該很強大，面試完後也沒等到任何通知，應該就是失敗。【鼎新】●職位：Software engineer●待遇(萬/月)：N-2●工時(小時/日)：9~10?●面試流程：筆試→資深工程師面試→主管面試﹝複試﹞鼎新在 ERP 界頗有名氣，所以小弟也去他們那瞭解了一下，筆試的部分先上機考智力測驗，之後小弟就被帶到軟體研發部門那去考 Java，考完後有兩個工程師來跟我面試，人都頗 Nice 的，讓我緊張的心情放鬆不少，後來回家後有等到第二次複試的機會，但小弟考量自己的興趣可能不在這，所以婉拒了。【和碩 (Pegadron)】●職位：Java 系統開發工程師●待遇(萬/月)：N-1●工時(小時/日)：9●面試流程：筆試→資深工程師面試→主管面試﹝複試﹞在板上有名的皮卡丘公司，進去到大門口還真的覺得蠻氣派的，還蠻喜歡一樓櫃台的感覺，沒多久就被帶到 MIS 專用的面試房間，筆試考 Java 和資料庫，基本上也是考觀念，有寫過一些 Java 的程試應該試都能過，之後跟兩位資深的工程師面試，介紹他們大致上的工作內容後，就散會等通知，大概等了一個禮拜通知我有第二次面試的機會，原本以為沒機會了說，後來再跟他們主管面試，主管一邊問我問題一邊在寫東西，不知道在記錄啥，可能是在對小弟的應對進退評分之類的。二次面試後隔幾天，還收到我的教授的信，似乎人資去連絡我的老師，想瞭解我在學期間的表現吧！後來有接到他們口頭錄取的電話。【奇鋐科技 (AVC)】●職位：MIS●待遇(萬/月)：N+2●工時(小時/日)：8~9●面試流程：筆試→資深工程師面試→經理面試這間公司位於五股工業區內，主要是做散熱的東西，例如：電腦風扇、散熱片等。到那邊人資小姐蠻親切的，挺著一個大肚子，倒了杯水給我，就開始說明上機考的流程，考的是英文，跟多益差不多，之後再考專業科目，題目蠻多樣化的，第一大題考名詞解釋，之後的考題內容包含 HTML、JavaScript、資料庫等，考完之後走進三個工程師跟我面試，過程中也是一整個輕鬆，後來他們就要我再等一下，直接找他們經理來跟我面試，跟這位經理也是聊得蠻不錯的，他也肯給我期望的薪水，很希望我加入，讓我有點心動，不過我的興趣還是不在 MIS，所以我之後用Email 向他們婉拒了。【Career 就業情報中心】●職位：程式設計師●待遇(萬/月)：N?●工時(小時/日)：?●面試流程：經理與資深工程師面試當他們人資跟我說他們是 Career 就業情報中心的時候，我以為是有其他公司有職缺要找我去面試，沒想到就是他們自己有缺，整個過程很簡單直接就跟他們的經理與資深工程師面試，工作內容大概就是要開發一些網站功能等。公司基本上是微軟的 Solution，所以用到很多「打內特」語言，另外他們的外訓基本上都由公司出錢，也不用簽什麼約定，除非金額非常非常大﹝好幾十萬那種﹞才可能會簽約，其他基本上他們經理說可以就能到外面如：微軟等地方上課。面試完後似乎他們也覺得我沒有什麼興趣，之後就沒再聯絡了。【行動智慧、愛歌科技 (ANSA)】●職位：Software engineer●待遇(萬/月)：N-2●工時(小時/日)：?●面試流程：筆試→主管面試這兩間公司規模不大，不過做的東西我有興趣所以就去了解一下，行動智慧這間公司專注研發『Mobile Information Platform』行動化資料擷取平台，另一間愛歌是做電子產品的研發。行動智慧的筆試大致上是考 Java，試題難易度我是覺得剛好，若有一些底子的去寫應該都覺得不難寫，之後與主管面試，由於我是應徵 Android 的 AP，主管就表明他們最主要的還是在做 Windows Mobile 的部分，所以會碰到這塊，我是沒排斥，不過主管似乎是想找有經驗的人，所以之後就沒有和我聯絡了。愛歌科技的考題有點讓我傻住，題目大部分都是考 Linux 的觀念，Java 大概占兩題吧！考的不是很好，因為對 Linux真的不是很熟，之後跟主管面試完，覺得他蠻想找人並能早點上班，後來面試完隔幾天有打電話給我，不過我沒接到，後來也就沒下文了.....【和泰汽車】●職位：? (只有說資訊類別職缺)●待遇(萬/月)：?●工時(小時/日)：?●面試流程：筆試→個案分析與面試→主管面試接到和泰汽車的面試，還以為是要去做行銷還是什麼的，原來是有資訊職缺，不過詳細的職務不清楚，似乎必須等到第三關主管面試時才知道。筆試當天先介紹公司的營運項目和福利，福利聽完後我真的還蠻想待在這養老的，考試內容有英文、圖形邏輯、專業測驗竟然是看報表然後看問題問的是什麼，再從中分析計算答案，附帶一提專業測驗要自備計算機。考完後大概等了一個禮拜，以為第一階段就沒過了，沒想到竟然還是有接到通知，可以參加第二階段，小弟目前是先答應，覺得還蠻有趣的，所以目前就是在等要去參加第二階段的測驗。【宏正自動科技 (Aten)】●職位：Software engineer●待遇(萬/月)：N+2●工時(小時/日)：9~10●面試流程：性向測驗→主管面試→英文測驗、人資面試﹝複試﹞→經理面試﹝複試﹞這間公司最主要的產品就是 KVM 了，相信有用過 KVM 的應該都知道這個牌子。當天面試時主管可能想要縮短時間，因為我之後還有人要來面試，所以原本應該是筆試的主管直接要我看題目，把我對題目認知的程度說明一下，之後就是一連串的介紹我的作品論文和，而主管介紹他們目前所做的 AP，聽完後我蠻有興趣的，後來等第二次通知，我主動聯絡他們，他們很快地就安排我進行複試，由於第一次太趕沒考英文就補考了一下，之後跟人資聊，過程也是一整個輕鬆，人資態度超級好，問蠻多關於我背景的問題，之後跟經理聊天，介紹我的作品論文後，經理也希望我到那上班，當天就直接給我 offer 了。【優必達 (Ubitus)】●職位：Software engineer●待遇(萬/月)：N●工時(小時/日)：9~10?●面試流程：筆試→主管面試優必達我覺得是間不錯的公司，專注於將雲端運算(運用在影音多媒體以及娛樂領域的軟體公司。筆試的內容考了 Java、JavaScript 等，我覺得考得蠻有水準的，有一定的難度，另外我對 JavaScript 不是很熟啊！所以我寫得有點痛苦，後來跟主管面試，主管問得很專業阿，問的東西還蠻細的，例如討論到我作品的某個方法是怎麼做的，若碰到怎樣的 Case 又該怎麼做，就一個出招一個拆招這樣子，有些後來我想想並不是說得很好，後來主管想要我先進公司到他同事下面學習之後再轉到他那，我覺得也蠻好的，因為他們用到地技術真的蠻多的，工作內容也蠻有趣的，工作氣氛頗好，因為都是年輕人，老闆好像才三十幾歲的樣子，當天我是晚上面試，還聽到另外一間的人討論的蠻熱烈的，之後隔幾天收到他們的 offer。【口袋移動 (PocketNet)】●職位：Software engineer●待遇(萬/月)：? (後來薪資有異動)●工時(小時/日)：?●面試流程：性向測驗→主管面試→經理面試位於內湖瑞光路上某間大樓裡的公司，口袋移動是透過我姊朋友的介紹後去的，公司專門開發手機的 AP，相信蠻多手機上都有他們公司寫的 AP。在做完性向測驗後，人資先給我一張紙，紙上面的內容是一段 Java 程式碼，大概有兩頁，要我先看，等一下主管會問程式碼內容是在做什麼，看完後我覺得不難，大致上跟讀取檔案與字串操作有關，主管後來就問我的作品及論文，之後開始介紹他們的工作內容，大致上研發分為 Server 端與Client 端，而他負責 Client 端的研發，在這邊可以學到蠻多手機 AP 技術的，因為他們要在蠻多平台上開發的，例如：Android、Iphone、Windows Mobile、Symbian等，談的過程都還不錯，之後有打電話通知我錄取。【結論】面試到現在有14間了，老實說真的蠻累的，從一開始面試的前三間，被打槍後覺得還蠻氣餒的，不過我還是告訴自己要越挫越勇，之後就比較有漸入佳境的感覺，在面試時真的不要緊張，緊張很容易在筆試放空，很容易在口試忘了要怎麼去介紹自己，忘了如何應對進退，另外要在每次面試後檢討一下自己的表現，盡量在下次面試中拿出自己的自信，還有如果自己有作品或論文的，做個 PPT 出來，印好後帶著一起去面試，以便在談及自己作品或論文時能輔助自己說明。若還不知道自己想找哪方面的工作時，其實真的可以多面試幾間，看看公司在介紹工作內容時是不是符合自己，從中瞭解自己的興趣，最後希望大家在求職過程中都能順順利利。--※ 發信站: 批踢踢實業坊(ptt.cc)◆ From: 111.250.207.250推 cloudfly:推一個...細心分享..推 grtfor:推清楚分享~※ 編輯: local9433       來自: 111.250.207.250      (09/15 23:48)※ 編輯: local9433       來自: 111.250.207.250      (09/15 23:52)※ 編輯: local9433       來自: 111.250.207.250      (09/15 23:53)推 bopa314:推一個推 suPerFlyK:推!推 hmanw:推推 cheei:你和泰跟我一樣~等超久以為沒過...不過我找到工作了0rz→ cheei:你也是3號禮拜五去面試的？那天資訊部門好像只有4個人推 dakkk:宏正→ local9433:回ch大，沒錯我就是那四個其中一個阿推 zeroade:原po真強 我連面試機會都沒有~~哭哭推 vincent3358:支持推 smafi:推認真文推 qaz8520:嗯嗯!  蠻不錯的分享推 ako:大推!推 DryBox:推推文自動更新已關閉
