2015各位先進你們好小魯為最南端國立科大學材料碩士畢業，小魯在面試前及期間非常感謝許多先進們在該板所分享的面試資訊及其心得，小魯今日也向各位先進們分享我研替時面試諸多公司之心得。-------------------------------------------------------------------------小魯自介大學排名:20%大學曾任幹部: 系會長、系上活動長、副班代、4屆班級公關、畢聯會班代表碩班排名:8 % (碩班人數較少)碩班曾任幹部: 畢聯會班代表多益成績 : 500-550-------------------------------------------------------------------------打工經驗派報生、薑母鴨、水餃店、7-11、演唱會工讀、跨年晚會工讀、菜市場ps:放寒暑假就是去打工-------------------------------------------------------------------------小魯原本抽到海軍陸戰隊，非常開心且期待進去受訓。有一天碩班朋友問我中科院有開放研替名額且今年面試人數很少，問我要不要嘗試看看。當時心中認為中科院是非常好的軍方研究單位，小魯心想如果能進去一定能光宗耀祖，而中科院就是我放棄從高中開始幻想的海陸特戰隊。ps:因碩班朋友家人在中科院上班，所以他告訴我人數不足的資訊讓我很心動。--------------------------------------------------------------------------＊研替面試公司: 中科院、凌巨、敬鵬、旭德、樺欣機械、鑫晶鑽、世界先進1. 2015/01/xx (忘了明確日期)中科院(龍潭)-(第一間面試的公司，我跟白癡一樣)面試單位 : 材料所面試過程 : 那天天氣超冷，我騎著黑色俏麗並穿著黑色西裝白襯衫，從我家騎到龍潭(15km)，邊騎邊發抖還差點迷路......(省略)。在等待進場面試前，外面排了許多座椅，我看了一同在等待的同學們，感覺大家都很厲害，我不敢鬆懈並仔細看熟自己所準備的履歷，以免在面試過程中忘了說自己獨有的特點。終於到我進場了，在進場前我大約計算其他面試者被問的時間(約10-15min)。一進場，映入眼簾的是5位面試官，我連坐都還沒有就一緊張就破口說  "good afternoon,every my dear interviwers"。 他們用中文我可以先入座，接續我又繼續用破英文說"Can I introduce myself ......."(我一緊張根本不知道我在說什麼)，因為我白癡行為而被中間最資深的面試官誤會成我是否為外國華僑或非本國籍人士.........心想天啊~~~我怎麼會白癡到這樣，之後緊張改講中文，連自己最熟的中文還講到有點外國口音。之後被問到碩士論文內容與大學在做什麼、曾任幹部、心中認為中科院是什麼樣的環境、另為何選中科院....等問題。被面試官所問的問題，我盡力回答並想多做補充，但他們沒興趣就是沒興趣.....唉。 自己的面試時間不到10min.....。碰巧我碩班朋友也跟我同一天面試(介紹我去中科院的朋友)，我問他有無碰壁，他說還好。我又問他被問到啥問題，他說他被面試官問到"你最喜歡什麼"，他竟然回答"女人"。我心想好險我比他成熟穩重多了，之後的兩周 "他拿到offer"、 "我感謝函"。面試結果: 滿漂亮感謝函.......2. 2015/02/xx 凌巨科技 (面試到最累也是學到最多的公司)面試職位 : 製程工程師面試過程 :第一天，我騎著黑色俏麗到八德華映廠區，離家約(&lt;5km)。我穿著黑色西裝與藍襯衫，進去公司要拖鞋並換室內拖。其中，大門小姐高瘦正，HR也很漂亮。面試步驟考英文、智力測驗及性向測驗，接續HR介紹該公司福利及其近況後，安排黃光製程的主管進行面試，該主管滿年輕的..........(省略)。接續又安排薄膜製程的經理進行面試，該經理聲音渾厚、博學多聞教了我超多，因現在已經晚間7點多了，經理詢問我是否願意二面。(面試時間pm3:00-7:30)第二次面試，薄膜經理帶我去看場區並介紹許多製程與機台，也是因此機會我才更進一步了解半導體的製程在幹什麼...(面試時間 pm3:00-5:30)。離開前經理希望我與黃光主管再進行一次面試......第三次面試，此面試都在與黃光主管聊天，我還白癡問他在該公司可以多久年薪破百.....。最後HR問我錄取意願等等，被中科院放鳥後，我當然1000%想進來，且越快越好。(面試時間pm3:00-5:00)ps: 與該公司面試時間前，我都在菜市場打工(am5:00-2:00)，所以我都跟凌巨約pm3:00。另外在此公司面試時，經理與主管希望我多去其他家公司面試，我抱著可能被放生的心態而去投了其他間公司，如日月光、欣興、世界先進、敬鵬、穩懋....桃園 ，龜山許多公司都投，為了讓自己被看到，而辦了三家不同的人力銀行帳號，並對好幾間公司重複投了好幾次....面試結果 : 等待2個月後，offer get 41K*14.5 (38K+勞健保)3. 2015/03/xx 旭德面試職位 : 高級工程師面試過程 : 地點在湖口，我開著家裡的馬自達前往面試。因我學長在裡面上班，所以他有告訴我要準備的功課。 面試步驟考英文、智力測驗...等。值得一提的是，我面試的陳經理很細心地告訴我製程工程師的工作內容，並跟我介紹SPC工具等...(很受用)。陳經理細心教我後，也不忘酸我的碩論了無新意，最後他也告訴我酸我的碩論是為了看我的反應如何。面試結果 : 等待1個月後，offer get 蜜月期38K，蜜月期後42-43K*14。4. 2015/03/06 敬鵬面試職位 : 製程工程師面試過程 : 地點在南崁，面試人數超級多，面試步驟考英文、智力測驗。過程就不多加贅述了，考試內容google很多。面試結果 : 等待1個月後，offer get 33K*15。我算過月薪含常態性加班費可達41K。5. 2015/03/xx 樺欣面試職位 : 機構工程師面試過程 : 靠近桃園高鐵，從場外看滿漂亮的一間公司。面試步驟HR聊天、考機械力學、英文。其中，在考機械力學時，我完全不會寫.....大一物理忘光、再來就是我沒修過機械力學........。因為我會2D的AutoCAD，但我不會看三視圖....。我心想我根本來當炮灰的。唉.......面試結果 : 副理覺得我滿上進的、希望我從基層做起慢慢學，之後就口頭offer，但我婉拒了，因為我覺得我自己有夠爛，什麼都不會。離開前並常感謝該公司主管與副理願意花時間跟我面試。薪資大約3xK有保14。6. 2015/xx/xx 鑫晶鑽 (日期忘了)面試職位 : 研發工程師面試過程 : 地點在竹科廠區，該公司漂亮乾淨。面試步驟去旁邊小桌考英文，接續與HR與主管面試，由於該公司我興趣不大，且亦是該公司主動邀請面試，我就直接前往了，有上沒上心態普普，不過也學到一些長晶的知識。面試結果 : 無聲卡7. 2015/04/xx VIS-三廠 (日期永遠記得，但不方便說)面試職位 : 製程工程師面試過程 : 地點在南崁，進該公司面試前，我從過年到邀請面試前投了8次履歷。在一間小辦公室進行面試，面試人數多、人資漂亮、主管高帥。面試步驟考半導體四大製程、英文、問卷題。面試這間公司時，我抱著可能會放棄上面所有得到offer的公司，VIS我真的很嚮往也很喜歡。所以在面試時，我很冷靜並將所有面試所得到的經驗與了解，在HR與製程主管的問答過程中，小心且謹慎回答。其中，HR主管曾說到如果當天面試若有認為回答的不妥或不完善，可於回家後播電話再多加贅述。也因這句話，我面試結束後將當天面試過程回想了一遍，並於隔幾天待HR主管有空的時候，重新再說給他聽。我非常感謝這間公司給我機會去面試，也非常感謝主管的耐心。面試結果 : 研替第二階段前一天我得到offer， (N-0.5)*16。 而我也選了這間公司，當我人生的第一份工作。ps
