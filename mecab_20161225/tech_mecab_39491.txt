Ltd.）以及德國一家工業技術暨系統研發廠商3S-Smart Software Solutions GmbH，所進口至美國境內供應販售之工業機械控制系統及相關應用軟體等產品，侵害其8項美國專利。本案系爭專利共計8件，皆是Rockwell Automation自行研發的技術，涉及工業自動化控制系統及用於設定、執行前者及視覺化監控該過程之相關程式軟體，簡述如下：1. US 6,675,226「用於工業用控制器應用程式介面之的網路介面」（Network interface for industrial controller providing application programmerinterface），2004年1月6日獲准，優先權日期為1998年11月17日，發明人為Suresh Nair、Michael J. Gilson。本專利發明涉及一種平行連接不同多重工業控制器應用程式介面之多重網路介面。2. US 6,816,817「具有即時監視的網路控制系統」（Networked control system with real time monitoring），2004年11月9日獲准，優先權日期為2000年9月28日，發明人為Kevin Retlich、Jinghui Luo、Dave Blair。本專利發明涉及一種同時具備即時監視功能之網路控制系統。3. US 6,819,960「工業用控制器自動介面」(Industrial controller automation interface)， 2004年11月16日獲准，優先權日期為2001年8月13日，發明人為Jeffrey A McKelvey、Mike D'Amico。本專利發明涉及一種可透過自動化介面來溝通不同工業控制器之系統。4. US 6,978,225「具有即時監控的網路控制系統」(Networked control system with real time monitoring)，2005年12月20日獲准，優先權日期為2001年8月13日，發明人為Jeffrey A. McKelvey, Mike D'Amico，原專利權人為洛克威爾軟體公司（Rockwell Software, Inc.）。本專利發明涉及一種具備即時監控能力之網路控制系統。5. US 7,130,704「工業用控制器自動介面」(Industrial controller automation interface)，2006年10月31日獲准，優先權日期為2001年8月13日，發明人為Jeffrey A McKelvey、Mike D'Amico。本專利發明涉及一種可溝通不同工業控制器之自動化介面。6. US 7,650,196「具組織結構化呈現層之生產監控系統」(Production monitoring and control system having organizationalstructure-based presentation layer)，2007年4月4日獲准，優先權日期為2005年9月30日，發明人為Kevin G. Gordon、Clifton H. Bromley、Eric G. Dorgelo、Douglas J. Reichard、Marc D. Semkow、Shafin A. Virji。本專利發明涉及一種使用於具接收組件、並可接收用戶請求來生成所需人機介面之系統的相關軟體及方法。7. US 7,693,585「在自動系統中啟用物件導向能力」(Enabling object oriented capabilities in automation systems)，2010年4月6日獲准，優先權日期為2004年9月30日，發明人為Michael D. Kalan、John J. Baier、David W. Farchmin、Randall A. Marquardt、Richard A. Morse、Stephen C. Briant、Sujeet Chand。本專利發明涉及一種在工業自動系統中啟用物件導向能力之方法。8. US 8,799,800「自動化用戶介面生成系統」（Automatic user interface generation），2014年8月5日獲准，優先權日期為2005年5月13日，發明人為Gavan W. Hood、Ralph Kappelhoff、Kenwood H. Hall。本專利發明涉及一種可應用於工業自動化環境、仰賴可編程邏輯控制器來接收、處理及輸出層次結構化資料之用戶介面生成系統。原告洛克威爾自動化公司於訴狀中，具體指出被告3S-Smart Software Solutions公司之CoDeSys工業自動化軟體以及研華科技之WA-CU系列箱型及電腦控制器和ADAM-5560CDS工業電腦侵害前述專利。研華科技因使用3S-Smart Software Solutions公司之CoDeSys軟體而成為被告。原告要求ITC發佈永久排除命令，以禁止前述產品進口販售。本次關稅法337條訴訟外，洛克威爾自動化公司亦於美國德州東區聯邦地方法院控告3S-Smart Software Solutions侵害相同專利（共兩案，Rockwell Automation, Inc. v. 3S-Smart Software SolutionsGmbH，2:16-cv-00869，2:15-cv-01543）。從過去訴訟記錄，洛克威爾自動化公司過去以案件被告身分居多，然自2013年來逐漸積極主張其專利權，控告對象以德國工業器械暨軟體製造公司為主。本案原告洛克威爾自動化公司，於2001年自母公司洛克威爾國際公司拆分成為獨立個體，之後陸續收購許多小型企業公司。洛克威爾自動化公司名列財富雜誌《Fortune》全球前500大企業，總部位於美國威斯康辛州密爾瓦基，在全球超過80個國家設有據點，員工超過二萬人，主要提供工業自動化、電源、控制及相關資訊服務，其產品包括可程式邏輯控制器、輸入輸出模組、人機界面、工業安全產品、變頻器、軟體、電磁開關、過載繼電器、感測器及軟體等。本案被告台灣廠商研華科技，成立於1983年，總部位於台北市內湖區，在全球21個國家設有據點，員工約八千人，主要針對企業及系統整合商提供整合性網路技術、電腦平台及客製化產品方案，所製造產品涵蓋工業自動化、產業及通訊用電腦、嵌入式電腦、網路電腦相關產品等，為出自台灣之著名智能系統品牌。研華科技雖有不少在美專利侵權訴訟經驗，然本案為其首宗關稅法337條專利侵權訴訟。本案被告德國3S-Smart Software Solutions公司成立於1994年，於義大利米蘭、中國北京設有直營據點，員工數約140人，並在10個國家有其產品供應或技術服務代理商，全球公司企業客戶超過400家。3S-Smart Software Solutions之主要產品為CODESYS IEC 61131-3工業生產自動化軟體及相關獨立系統平台，該產品主要用於多領域工業自動化生產中相關智能機械裝置之控制、相互整合及應用上。本次337訴訟及前述德州東區聯邦地院兩案，為該公司首次在美遭控侵權。──遇上ITC337事情就沒那麼容易了!--※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 192.83.171.120※ 文章網址: https://www.ptt.cc/bbs/Tech_Job/M.1473324048.A.89F.html推 chuegou: 只看紅字感覺很容易踩到→ wisdom: CoDeSys如果被告倒了，一堆用這套軟體的公司就挫著等了推文自動更新已關閉
