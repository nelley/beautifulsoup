Soft_Job標題: [心得] 面試經驗時間: Thu Nov  1 01:35:30 2012回到台灣20天了，經過一連串十幾家的面試，總算塵埃落定。今天要上班了，再幾個小時之後就有工作了（哭）。就來分享一下經驗吧。陸陸續續在美國有以電話與臺灣的公司面試：國家同步輻射中心  Software Engineer新竹的財團法人公司，有個很大的槍做實驗，這職位是要記錄每秒各種不同圖　　　　，並且以平行和分散式處理把資料消化處理。有吃到offer，但是不是太好看態度不是太好，但是基本上是個鐵飯碗。CitiBank          Financial Analyst(Developer)這家之前文章介紹過了，基本上因為我太晚回臺灣，但太早面試，沒辦法等我。做推測一堆金融的網站和軟體。要面試請準備C/C++/Java, GC和基本程式概念要念熟。優訊軟體科技      QA Lead Engineer這家之前文章介紹過了，博亦類型公司，馬來西亞商（還印尼商），主要是　　　　做一些賭博軟體網站供國外使用。這家，該怎麼說呢。應該是符合他們要求，可是開不起我想要的薪資。不過，面試後也都沒下落，基本上對你們公司沒興趣，我又何必去面試呢。Cadence 益華電腦  Lead Engineer做IC設計軟體和硬體相關的公司。有情有義的一家公司，少數在美國時就願意我接受就直接出offer給我，　　　　價碼只能說，也是很有誠意，只可惜，我還是以台北為主。　　　　特別提到這家公司當時我做了點研究，如果有碰過PSPICE這套軟體，就是他家的　　　　，這方面他是屬於世界第二名。小情報是我朋友和我說過，其實他們很多人，都　　　　要去台積電裡面做事和協助，聽說位子不太好，不如直接進台積電比較划算，新　　　　水來說應該台積電拿紅利後還是比較好。Netbase           QA Engineer做資料探勘、資料分析等等來完成Social Network分析一類的（沒有瞭解太多）會調動國外的人和你面試，基本上我當初事先作一份他假設好的網站pdf，然後　 　　 規劃出他的測試方法、那裡有問題之類的，等通過這關，會有人和你面試，當初　　　　似乎他們自動化測試還不太多人使用，所以當時面試我的軟體工程師似乎知道很　　　　少，後來美國Director想要把我在美國先開始遠端工作，不過因為身份關係，當　　　　時Netbase尚是臺灣公司辦事處等級，無法收我，只好作罷。（註：現在已經是子公司等級，不過只缺NLP的人了）聽說軟體工程師會是上機考，上機考前在家也會有一份考題。以後還對他們QA有興趣的，準備方向是Selenium Webdriver、TestNG、Linux趨勢科技          Senior Software Egnineer*3之前有提過了，基本上，我直接進到最後一關和幾位主管談，但是談完後就　　　　接感謝函了，原因我也不知道，大概是我還不在臺灣？或是覺得我很弱。鴻璟科技          Software Engineer寫韌體在硬體上防毒和安全性的一家，很重視演算法和C的各種概念（不是C++），開始會寄一封信給你解題，我回台灣沒有立刻去新竹面試，所以後來本來以為沒希望時，就開始投台中新竹的面試，結果就有收到Offer，就沒有繼續面試。有興趣面試建議準備C的指標、指標函式、記憶體空間和演算法。A10 Networks      Software Engineer有福利的好公司一家，做雲端、Switch、網路最佳化、網路各方面技術的一家，在臺灣算剛開始一小陣子，但也不算是startup，需要非常即時和符合技能的人，有興趣可以去104查看看，基本上問題都會包含著他上面的範圍，所以幾乎要能技能群完全符合，可惜我都沒有接觸那些東西。沒能面試通過。回來臺灣後，又開始奔波面試：訊連 Cyberlink   Core Team Software Engineer做PowerDVD/PowerDirector/Power...的純軟公司，面試會先給你英文和智力測驗，再給你軟體考題，就是很多C指標的* ++ -- ()搭配的題目，還有些演算法問題　　　　，覺得算是頗簡單的。之後就和主管面試兩關。後來有打電話問我要不要來，但是　　　　我說我還在等某一家公司，他說他過一週後再打給我。群暉科技         Software Engineer做NAS的公司，我覺得當天題目算是對我比較好，開始有出簡單一點，不過是說我　　　 當時時差算還沒調回來，此外演算法都遇到我沒太能想出來的題目，兩個小時多我就被請走了QQGarmin          Software Engineer做GPS的公司，當時寫完英文和性向測驗後（這邊英文和紅茶店考接近），不過很　　　 有趣的是，這其實不是Software Engineer的缺，而是手動測試的缺，所以就不歡　     而散，不過這是我見過對應徵者最好的一家公司，還可以搭計程車去，他會給你全額補助，以門面來說，這家HR是最高規格!!!臺灣雷技        Sr Software Engineer一家在南港育成中心起來的Startup，主要承接美國密裡蘇達州的業務，創造一些　　　 行動平台和網站，之後也會自己接收一些不同CASE，而在台灣有和一家公司合作　　　 ，不過很沒禮貌，打手機請人去面試，不過問他們做什麼的時候，他說你來我在和你解釋，過去後問很基本的程式問題(MVC, .net, C/C++, Android ...)。紅茶店         做紅茶的紅茶店 最沒有態度的一家公司。開始提早過去還沒開門。後來坐在前面櫃臺那邊約半小時，總算有HR來接待了，丟我進了一個小房間。椅子做下去靠到玻璃很不舒服，前面桌腳很大，卡住腿後，更不舒服。來談談他們寫的英文測驗軟體好了，題目有一部份是跟Garmin一樣，很弱，連題目都有錯字。更不要說不人性化的聽力測驗，聽完後只有一秒鐘選答案，這哪招阿？還不能往回選。(註：四年前考過TOEIC已經在900分以上，Garmin的　　　 我也拿900左右，TOEFL 100分: 聽讀各29分{滿分30分})　　　 做完智力測驗和英文測驗，打電話後叫HR後，在狹小的監獄又待了10分鐘才有　　　 人來，是說我這時沒水喝也沒上廁所。之後，HR又讓我在外面待了10分鐘，才　　　 拿了一個紅茶店機密文件夾給我，叫我從這個building自己走幾個block到新　　　 的紅茶店大樓。然後我又在新的紅茶店大樓站著等了十來分鐘。　　　 接著被HR領到面試室，門沒有關，聽到主管在訓HR："這個人學歷不錯，ＸＸ　　　 大學嘛，OO大學嘛，可是你看他經歷，沒有Driver經驗"（所以上面寫C/C++　　　 兩年經驗"或"Driver經驗是什麼？？？），主管繼續說："妳也要給他做C/C++　　　 測驗阿，這些事情為什麼你都辦不好，而且他比較像是Application或是其他軟體部門的缺才對。"　　　 後來主管進來談一談，說會把我履歷轉給其他部門。　　　 對了，到最後我還是沒有上廁所。(加註：我覺得他們智力測驗是最難的一家)臺灣雅虎        Sr Software Engineer &amp; Software Security Engineer在美國先和一個主管面試過，後來回臺灣以後在做第二次面試，這家有個HR，　　　 不管問什麼事情或做什麼安排都超混亂，我基本上對這家HR門面抱有極大質疑。我是面試EC（電子商務）部門的，這裡現在情況非常有趣，因為CEO換掉了，而CEO決定要看過世界上所有Yahoo錄取的人的履歷再決定他可不可以上任，所以　　　 不但面試很久，之後還要給CEO看過和同意，粗略估計大概要五六週。我是覺得環境不錯，有免費飲料點心和遊樂機。面試題目則是，他們覺得用很難演算法考人會嚇跑人或問不出所以然，所以　　　 他們會以很簡單的演算法題目問你，看你程式熟練度，會寫程式的不必太擔心。　　　 可是理論可能得看熟一點，考很多程式方面的理論。QNAP            Software Engineer一樣是作NAS的，簡單的兩題題目純粹是看看你懂不懂程式和熟練度(C)。　　　 沒什麼特別好提的，相較群暉的地獄，這裡題目簡直都是天堂。Mozilla         SQA Engineer (Developer)來臺灣做Firefox OS的，本來是想要進去面試Software Engineer，不過似乎很　　　 看中經歷，所以當場被轟不夠資格，面試時就是談你的經歷，他會針對經歷提出　　　 疑問，會簡單的看你程式能力（白板或電腦）。環境看起來和Yahoo比較類似，　　功典資訊       Software Engineer做CRM的，總之就是幫客戶建立品牌，然後管理客戶，此外他們缺少分析大量資料　　　的人，所以再找雲端和大資料處理的人。基本上都是去問你經歷，然後沒太多技術　　　題，最後要見CEO，他們CEO是很厲害的人，我覺得有朝一日，這家公司會坐大的，現在缺會英文和有技術的人，見過他們CEO就會覺得有種這家公司充滿希望的感覺。　　　建議去面試的人準備好他們有什麼產品和服務，定位一下自己可以幫助他們公司什麼。最後，我覺得他們是真美式管理，但是帶有一些台式的人情味精雲科技       Software Engineer精誠科技子公司，雖然說是子公司，其實只是拿精誠的錢做事這樣，老闆都是從美國回來，他們做的雲端儲存，但是不是NAS，我有點忘記詳細內容，但是他們要　　　做Scalibility比NAS好的儲存裝置。也算是start-up，我對這個是還蠻有興趣的，　　　也覺得這應該會比較偏美式管理。不用太多特別準備，只要把經歷和學到的東西講清楚。TutorABC      Software Engineer咦？什麼，這家不是教英文的嗎？沒錯，不過他們現在積極致力於很多技術帶動公司的策略，所以他們再找各方面的軟體人才，尤其是處理Big Data，還有資料　　　探勘，還有一些資料分析，基本上就是做出能夠推薦適當的課程，能夠分析使用　　　者資訊和行為，以改進系統，另外還要讓系統的Scalibility提高，因為目前正　　　準備進軍其他國家，估計在2015會上市美國NASDAQ。這家公司有種特殊的氣息，其實還蠻吸引我的，佈置有品味，環境類似Yahoo/Mozilla，我覺得這家公司可以坐大的，格局和氣勢看起來不錯。　忘了說，要面試這家公司請準備基本演算法、作業系統、分散式系統、平行處理、Thread/Process...問題不難但很切重要點。需要寫申論題和智力測驗，另外　　　要提供FB或email帳號，建議...你知道怎麼做的其他數家公司族繁備載，以後有想到回來補充。--攝影旅者
