2015大家好，我先自介，小弟不才後段國立學、前段國立碩，光電背景。退伍至今將近兩個月，找工作重心放台中或台南，新竹一些。工作找得實在很茫然，面試得很無力，履歷投得都麻痺了。面試過一些公司，以下簡略報告：六月 / 南科 /  台GG  / 黃光製程 ：好人卡。八月 / 竹南 / 小公司 / 太陽能RD ：無聲卡。九月 / 后科 /  美光  / 黃光製程 ：好人卡。十月 / 竹科 /  隆達  /  光電RD  ：好人卡。今天 / 草屯 / 小公司 /  LED RD  ：口頭offer。小弟沒什麼雄心大志，本想找個傳產穩穩做，但現在連傳產都慘兮兮。LED照明現在看起來的確也像傳產了。昨天，對就是昨天，接到這公司的電話邀請面試，直接劈頭問今天下午或明天下午有空嗎？心想：這公司也太缺人了吧！從台中坐客運到草屯，約一小。提早20分鐘到達，到了面試時間1點半，出現一位小姐，把我們帶去，包含我似乎4個人。在裡面等了20分鐘，核對資料發現沒有我，因為...他們是工研院的=  =我被帶去別的地方啦！好了，插曲結束。終於，他們公司的人資下來把我領走。在會議室等了一下，出現一位30多歲的人，我以為是主管，結果是公司創辦執行老闆啊！以下簡單敘述當時面事情形，A為我，Q為老闆。Q:先自我介紹吧！((我就用很聊天式的方式自我介紹，說了一半我準備的內容我就被他拉走了XDQ:你是我破格那麼快就約來的人，本來有固定時間跟固定流程要走，我通常都是第二關。((突然有種被好好對待的感覺了。Q:你的履歷讓我看見不只你的能力還有你的為人特性。我們公司要得是沉穩踏實個性的人。輪到我發問(感覺我有點像記者在問=  =)：A:我想請問您在這邊工作多久了？Q:我就是這邊的老闆(70年次)。((靠北，超尷尬！A:這產業被稱為慘業，您怎麼看？Q:那是他們策略問題，其實這市場還是很大，我們做到全國去，台灣有大部分照明都我們這邊提供。A:請問現在都講求快速，您怎麼看？Q:我們這公司講求得就是穩定踏實前進，你的背景就讓我覺得很合適，雖然我們公司不會大賺，但一步一步還是可以累積的。其它聊得總結：在荷蘭、西班牙、大陸、台中及桃園都有設廠，行銷到80幾個國家。今年預計股票上市，也剛接到國內大案子。試用期3個月，N-15。試用過就三年一簽調整為N，年薪計，N*12。每年升等加薪，分紅、獎金另計。(N&lt;GG價)老闆有提到，他們8成的員工都會被高薪挖走，但他還是堅持願意訓練新人。面試結束很爽快就問我什麼時候要來報到。我聽到嚇一跳，這公司果然很缺人XD((不是說穩重踏實嗎XD我說我想考慮一下，可能年後了。他也很爽快說，沒關係！我不壓你期限，給你我特助名片，隨時想來就打給我特助。我想如果我投身這行業，以後出路真的會變很窄，但小弟實在沒什麼鴻圖大志，是有在考慮。所以我跟他說年後，我想再試看看其它路。真的很感謝這位老闆的賞識，回到家後傳了一封email給他道謝。以上。如有冒犯請見諒。--※
