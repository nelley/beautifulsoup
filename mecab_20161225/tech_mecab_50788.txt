發文，今天的遭遇實在有太多話、疑問、苦水要講了，先感謝大家耐心看完我的苦水文@@先介紹自己的背景，我不是CS背景出身，沒受過專業訓練，也沒證照，對軟工相關的知識大概是幼稚園程度。我大學主修物理系（中壢比較偏僻的那間），一直到碩班都還是唸物理所，碩班開始靠計算物理吃飯，領域偏量子統計力學，要用電腦模擬的方法來研究多體物理系統的一些特性，例如相變情形和能階。在我們實驗室，程式語言工具主要靠 Fortran/C++/Python 還有一點點 Java...（當然也是有其他實驗室使用 MATLAB...），我在高中時有接觸一些 C 的底子，所以至少在基礎的運算邏輯流程控制都還是可以通的，Fortran 看了2週的範例就開始寫東西來計算研究，C++ 也是邊看範例邊摸，當然我知道這種學習方式沒什麼系統...一開始主要是這兩種語言在使用，由於我們要處理的模型尺度很大，一般 PC 跑起來很費時間，指導教授就要我們研究一下多核心平行運算的用法，把工作切割後丟到機房運算叢集上面跑，所以我學會了MPI 和 GPU運算（CUDA）；至於 Python 和 Java 則是因為指導教授的其他合作者有在使用，而他們寫好的一些功能模組我們覺得很實用，指導教授就要我們順便研究一下，所以有一段時間我也順便學了一些 Python 和 Java 的寫法，但不是主力就是了。另外由於整個碩班生涯都在跟機房的伺服器為伍，所以 Linux 系統的基本架設、維護、故障排除、簡單腳本撰寫的技能也不知不覺學會了。我畢業後一直都在學術單位從事數據分析、資料科學相關的研究助理工作，年初的時候突然動了念頭想要往業界試試，於是做了很蠢的決定：從原單位辭職，然後改到我家附近的國高中文理補習班當兼職解題老師，為的是白天能有更多時間應付面試...我求職設定的範圍很廣，只要跟程式寫作、軟工有關的工作我都想 try...到目前為止，會找我的大多是韌體廠（大概看到技能欄位有 C++ ?），少部分是證券業金融業的資訊部門（元富看到我會 CUDA 主動來找我），但基本上第一次面談後都還是發無聲卡，大概對他們來說我是白紙，會顧慮吧？今天要說的主角甘霖（台北博愛路那家）是第一家有回應我的軟體業，看介紹說是做票證以及信用資訊查詢系統的公司，客戶都是銀行。收到通知一開始還有點感動：哇！！！他應該算是軟體業吧？第一家軟體業理我了誒！不知道是不是我太嫩，簡單來說面談過程他都會挑你沒辦法反駁的地方來打壓，例如我在業界是白紙的事實，雖然談話內容好像很誠懇，但薪資開出來完全不是那回事...我原本期望至少可以給我跟科技部計畫（學術單位）持平的起薪(碩士36000)，但猜猜看他開多少？.....26K.....好吧～我知道自己是白紙也不是CS背景出身，但這會不會有點誇張？對方還問我接不接受加班，開這種起薪還要我接受加班？說觀察期3個月，之後會調薪，我也不想問對方幅度大概多少了...離開後猛然想到另一件事情：今天是光復節，勞工應該是放假吧？所以他們違反勞基法了...chrome 外掛「求職天眼通」竟然沒有顯示他們的違規紀錄XD請各位先進幫忙分析看看，我的要求會不會太超過白目，還是這間真的太欺負人？還有沒有哪些相關工作或公司比較願意接受白紙？謝謝。--◤     ◥ ◢ ◣傑米，炸掉它吧。  ⊙▁⊙─   ─⊙▂⊙ 碰到問題，用C4就對了！╰  ∕皿﹨    ◥皿◤ ╯◥█◤◢ ◥ ︶◤Adam Savage ◤  ︶    ◥◤  ﹨▼∕◥ James
