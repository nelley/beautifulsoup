版友常說這間工司文化很糟 還一堆人想跳坑之類的: : 可以跟各位分享一下我看到的: : 1.組織一直改組 大概3~5個月就會換一次老闆的確常有組織變動，我的單位並非每次都有換老闆: :   改組的時後各單位之間職務不太會交接: :   舊單位的人跟你說組織已經改組了 去找新單位的: :   新單位的跟你說這個之前不是我負責的 去找舊單位的: :   推來推去結果找不到負責人的確有這樣的現象，交接上有落差: 本人在三口當PE超過五年，"平均"兩年換三個老闆: : 2.如果是個不會踢工作的人 那你接工作的時間會比實際作業的多: :   就算不是你負責的製成也會找上你 原因是因為不知道找誰負責: : 3.設備歸產線管 跟OP一樣 所謂的工程師只有指製程 設備是不能被叫工程師的設備是歸設備課管理，製造沒有權利管設備，但產線設備需配合製造生產，例如輪休等。: :   所以製程責任很大 不知道為什麼有些應該是設備的工作 負責人卻掛製程: :   像是新機台導入 設備更新等...: 我的單位設備一樣稱作工程師，主要工作是維護保養設備: 因為設備更新通常需要參數的調整: 所以最後的責任大部分都是掛在PE新機導入負責人掛PE很正常，我的單位即是如此。新機導入流程該怎麼做，改機參數，參數怎麼設定，表單要紀錄什麼，抽檢頻度，buy off時機等等，不就是PE去制定流程嗎？？: : 4 工司對員工不太好 很苛刻 像製造單位會要求一個人顧好幾台機台我想知道哪一間公司會一台機台配一個人。人機比是IE算的，如果是1:5那就是一人顧五台。當自動生產時，人都在等機台生產結束那些等待時間是浪費的，所以才會有一人多機的情形若是手動作業就有機會一人一台... 這個情形一個人顧兩台以上會沒有效率和產出。: :   所以在這種氣氛下製造單位會刁難工程師 尤其是菜的: :   覺得工程師過很爽 偶爾進產線指揮他們做事就好: :   如果有客戶特殊需求的料，PIE會在工單上掛工程師的名子，: :   到站時會在旁邊確認製造作業流程: :   製造都會稱這種為"你的料": :   我常常被用很不客氣的口氣問你的料到站了 是要不要進來看: :   你再10分鐘不進來我就把你的料拉掉之類的: :   但那不是我的料 是客戶的 營收也是製造在收 掛工程師確認是因為有特殊需求: :   不過產線有仇視工程師的心理 遇到這種的他們都會刁難: :   機台有空間一定是以量產的料為優先...有空間跟人力在做"你的料": :   (跟他們解釋很多次 那不是我的料 不過產線好像聽不懂): :   我還曾聽設備組長說過"那個工程的料先擺後面 你先來做我這個": :   因為只要掛工程確認 該批料的良率就全由制程負責了: :   不管是OP摔料或設備有問題造成的問題 全都由製程負責: :   所以產線對要工程確認的料不會特別上心 (反正有問題工程師負責): 因為製造單位負責的是量產的單能不能順利產出: 工程的貨責任不在製造單位又會排擠量產的單: 當然生產線製造主管就不會特別去留意工程的貨通常工程批會有產前會議排出各站工程師，哪些需到場確認，治工具準備如何等，小PC會排上排程，製造會依照排程去生產，時間該給工程批就會讓出機台，但當工程批作業時間超出排程時間，製造就會反應PC，因為已影響後續量產排程。也常有外單位PE 或PIE，臨時就要產線配合做什麼，也沒有排程，製造哪來多餘的人力去做那些非正常流程的製程，又影響效率... 因此這類需求都會被否決，除非主管同意吧: 所以通常PE新人一進產線的第一件事就是買飲料請製造單位可能是單位文化，我都沒凹PE請客，沒這個必要吧，該是製造要配合工程那就作業。: 至於任何異常只要不是明確的指出是人為或機台的問題: 通常一律歸給PE正常是設備先調查異常原因，除非很明顯手OP做錯，不然都是需設備先調查，再將結果PASS出來，而不是什麼都PE: : 5.我那個處沒有四班二輪的PE 工程師白天除了要負責專案外: :   如果半夜有需要確認的工程批要留下來確認 這種事常常發生: :   這點我覺得很奇怪 因為一般科技業做專案的跟顧線上良率的多少會分開: :   產線是24小時的 卻沒有四班二輪的製程 要求普通班的24小時工作...: :   聽說有的處有42輪的PE 有的處沒有我的單位是有夜班工程的。: 我的最高記錄是約26小時，還好可以報加班不然就虧大了: : 6.能夠待3年以上就算超資深: :   有的站製程比較賽的 全課沒有超過3年的工程師(包括主管): :   也沒有人想升課長 因為升越高越賽: : 結論就是一間很操很賽的公司 大家都忙到口出惡言: : 氣氛自然不會太好: : 而且進去會發現有能力的人都跳去其他地方了: : 會留下來的都...不是很會推就是本身條件也沒其他地方可去: : 所以也不要期待進去可以學東西練功: : 是情都很簡單 有人檔著說他不做而已: : 也不要以為可以去學做人 裡面像董大媽一樣無法溝通的一堆: : 講人話他們也聽不懂: : 只要有辦法叫別人做事就贏了: : 但是有辦法叫別人做事的新人又有多少: : 通常都變成老屁股前就夭折了: 雖然已經從三口畢業，但是還是要說: 賽的地方還是有好缺，好的地方還是有賽缺: 如果想待台中工作，三口不見得是不好的選擇: 重點是除了一般的工作內容你真的學到什麼:
