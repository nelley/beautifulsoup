2011第一次在此發文，小弟不才，請多指教！大概從去年十二月開始投履歷過程中常常會跑來這個版參考各種資訊，如公司的前景、好壞、氣份、面式方向等等，四個多月過去了，研發替代役也差不多告了一個段落，因此想分享一些公司的面試心得。對了，常聽大學、研究所成績單在面試非常重要，小弟不才在大學貪玩，當公關辦聯誼、搞社團，所以大學成績一堆不及格甚至有零分，研究所也有被當的科目，等到開始找工作的時候，才擔心了起來....畢竟許多公司的面試都要看成績單，但是這幾趟面試下來，在學成績或多或少有影響，但只要老實的跟面試官說，其實都還是有機會的，其中有問幾位面試官，關於在校成績是否重要，大多是回我看這個人是貪玩，還是真的不會讀書，如果是前者他們會要，而判斷就在面試的過程中。所以成績不好的更應該努力去嘗試看看&gt;&lt;"小弟並非名校，從一開始對面試的生疏、緊張到一直到變得樂愛於面試，能夠與不同的主管對談，只要敢發問，就能學到非常多。(依面試時間順序)公司 : 康博科技 (未上市、電視盒、監控)地點 : 新店(統帥工業區)職務 : 軟體工程師過程 : 離我家很近的一家，因位於工業區，附近環境不算太好，但公司內部還算不差，考試智力測驗與程式設計(有加分題)，難易度為中下，加分題則為中，有一點陷阱題。考完主管就會來面試，因為是第一次非常緊張，但主管人超好，不斷告訴我不要緊張，在我無話可說的時候，還不忘提醒我，現在不表現自己就再也沒機會了，給了我很多重來的機會，過程中大概就是專業對談、問成績單、個人優勢、論文、介紹公司。接下來就是人資聊天，(個人特質)。大概快三小時第二次面試，一份英文問卷，另一位部們主管面談(長很帥)，主要以論文、專業為過程。最後公司氣份感覺普通、但人都不錯。薪資 : N-- (第二階段有補)  12N 無任何保證公司 : 技嘉科技地點 : 新店職務 : 網通工程師過程 : 沒投履歷，被莫名其妙找去考試，在某個星期日，由警衛伯伯發考卷，在一間會議廳考試，有很多人，考試的感覺很像再考研究所......難度中、C的部份印象考的滿機車就是... ...考完後就沒消息了= =薪資 : 不知道公司 : 晶訊科技(未上市、藍芽、WIFI)地點 : 中和職務 : 軟體/韌體工程師過程 : 時間是某天晚上七點，已下班時間= =...無考試，我特別準備筆電、自我介紹、然後聊一下專業跟介紹貴公司，之後就是將近兩個小　　　　時的哈拉...比如鴻海為什麼會成功、ＨＴＣ怎樣、某支股票ooxx。整個過程約兩小時半。薪資 : N (第二階段未補)  13N  好像有表示三年不調薪..　公司 : 宏達電地點 : 新店職務 : 軟體工程師過程 : 恩...感覺人資很不鳥人...飲料是柳橙汁，進去就馬上考試，有非常多份的考卷，邏輯/C++/C/英文。(跟板上的考古題差不多...但沒把觀念完全弄懂只是抱佛腳的話可能下場跟我一樣= =)分數依序是...100/4x/8x/4x...結果因程度太爛&gt;&lt;無面試機會，哈哈...非常慘的一天&gt;&lt;"只好默默背著筆電，人資還沒送我到外面&gt;_&lt;報完分數就跑了，..害我自己離開還卡在門口沒有員工證刷卡出去...只好默默等有員工離開才跟著出去&gt;_&lt;...感覺很差。薪資 : 無機會得知。公司 : 研揚(IPC)地點 : 新店職務 : 韌體工程師過程 : 這一天也是非常可悲的一天= =...考試內容很雜...名詞解釋、C、組語。有限時，有點寫不完...難度中，考得有點廣，很多東西忘記&gt;_&lt;，結論是空很大，原以為會跟HTC一樣下場，正要默默背走筆電時，主管就來了，一樣自介、專業、看成績單，會很仔細的問妳很多問題，包括成績單、論文、或抽問...最後還會給一小篇英文Paper，當場給五~十分鐘看完報告。最慘最慘的是...離開後機車被拖走...要去面式的要注意阿= =+那裡很難停。(大概兩小時半)薪資 : 不知。公司 : 英丰寶(未上市，純軟)地點 : 南港職務 : 軟體工程師過程 : 公司地方不好找，進去要跟警衛換證，但環境感覺很佳，一進去考一份JAVA...有限時間，我相信是正常人都做不完的= =加上JAVA有點不拿手，瞎猜不少，幾乎是觀念+陷阱的複選題...難度個人覺得難。考完後面試，公司要求準備簡報十五分鐘，聽說有三次面試...不過第一次完就沒消息了= =。面試的過程感覺也很差，簡報的時候面試官東張西望從沒往投影片看，真難過&gt;_&lt;。(兩小時內就結束了)薪資 : 不知，但面試官說比同業高很多。公司 : 四零四科技(未上市，IPC)地點 : 新店職務 : 研發工程師(進去才分部門的樣子)過程 : 慘痛的一次經驗，會先以電話面試。聽說福利佳、對員工好、(來自網路與主管口述)，不過經過辦公司內有看到不少單車停放，感覺公司氣份也很棒，最重要的是櫃台小姐也很正XD非常親切、超棒的。一開始先考專業與邏輯，邏輯雖簡單但滿機車的。難度算中下偏簡單，大多要自己設計程式碼。面試官有三位!(第一次碰到相當緊張)，其中一位表明是同校畢業的學長XD他是主要與我對話的，也因為這樣我幾乎沒辦法顧及到另外兩位面試官，，整場下來面試官親切中帶有嚴肅，一開始準備的投影片遭到拒絕= =，直接要我上去利用白板寫論文架構以及描述、自介(過程被小電)，後來回到自己座位，又被三位面試官分別連續問問題，而我幾乎都答不出來...或答不好...(慘)剩下的就是面試官介紹貴公司，也表明很賺錢?不缺資金，所以不上市，，有很多社團，在金融風暴仍然賺錢。然後問我有沒有問題，大概說一說就送到電梯結束了&gt;_&lt;(過程中學長人很好也很罩，只可惜自己表現太差)隔天被發卡XD薪資 : 聽說很不錯? 無緣得知。但主管提到研替視為一般正式員工。公司 : 融程電訊 (上市 電腦周邊)職務 : BIOS工程師地點 : 三重過程 : 公司營運整體不錯，氣份感覺也不差，櫃檯小姐非常正。一份性向測驗與專業考試，難度非常簡單。主管非常年輕，自我介紹後，簡單的問一下專業，然後主要問你對BIOS有沒有很有興趣，然後問離家近不近之類的，大概聊聊天不到一小時就結束了。薪資 : N+ (第二階段未補)  聽說14~16N? 每年內都會調薪公司 : 瀚霖科技 (未上市 無線網路、通訊)職務 : 軟體工程師地點 : 新竹過程 : 老舊公寓，入口只有一個小小側門，氣份未知，人資跟行政人員都非常親切，感覺裡面每一個人都很好，中間偷偷溜進去上幾次廁所(假借廁所名義偷觀察內部氣份XD)。兩份考卷，分別是網路、WEB。可擇一，考試內容可以說是大雜燴= =... 難度中上(其實空了一堆沒寫&gt;&lt;)。面試有人資、主管、研發主任，輪流問專業、聊天、介紹公司。感覺也是開不完的會議，不過裡面的人真的很親切也很好，不像之前板上文章說的很差。(也許是部門不同)薪資 : N (第二階段有補)  口頭 14~15N 會調薪公司 : 曜碩科技 (日商)職務 : 軟體工程師地點 : 台北市民大道過程 : 環境很棒，女生多，入口有一隻大鋼彈，很特別@@"。第一階段考試，聽人資說今天才開始找人來考試(真晚= =)，說我是第一位考這份新改版考卷的人，恩...考卷內容很難++(一堆要死背的題目)，有限時間，一大份JAVA及C，考完後等待考試擇優面試通知，大概是沒機會了，因為題目二十五題我只會三題。考後跟朋友討論，才發現原來有技巧...，知道技巧我想肯定能拿滿分...哀(時間不能倒流XD)薪資 : 未知(剛考完)公司 : 凌群科技 (純軟)職務 : 軟體工程師地點 : 台北西門町過程 : 智力測驗 + 面談 + 問卷，通過才有第二階段面試。智力測驗很多題非常煩= =...公司整體環境不錯，氣份也很佳，女生好像也不少，主要是接客戶需求來做專案的，但面試官強調有外派及因需求調職務的可能性。薪資 : 未知(剛面試完)老實說打到這頭好昏=
