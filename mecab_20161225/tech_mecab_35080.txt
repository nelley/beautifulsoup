Chrome 的插件，能在 104 和 1111 的職缺下面留言討論（可以匿名）。載點在這裡：https://goo.gl/OPqwLV假如不放心的話，我把 source code 都放在 github 上面：https://github.com/abalone0204/Clairvoyance然後以下是我認為有些問題可以跟大家討論的。1. 為什麼要做這件事？2. 為什麼人力銀行沒有留言功能3. 身為一個軟體工作者能為這個就業環境做些什麼------1. 為什麼要做這件事為什麼做這件事的原因其實蠻直覺的，常常看到 soft_job 版上面看到問某某職缺如何，或是問有沒沒有人去某某職缺面試的經驗。我就在想為什麼不能直接在人力銀行下面直接留言討論勒？而且這不是本來就應該要有的功能嗎？2. 為什麼人力銀行沒有留言功能(這裡就不討論好不好用這個主觀的問題 XD)之前有看到 TonyQ 在臉書上 出來的討論串：https://www.facebook.com/tonylovejava/posts/10209600345751168在探討現在台灣人力銀行的盈利模式，（裡面有很多有趣的討論）不過簡單做個結論，台灣的人力銀行是偏向對資方收費，而我們身為勞方其實某種程度上就是人力銀行商品的一部份。如果能提供好的媒合，這其實沒有什麼問題，但用加大資訊不對稱來滿足使用者（資方），就不是個正向的循環了，而這偏偏是現在檯面上人力銀行正在做的事情。引用 TonyQ 文中的話：「104 在沒有不同於（資方）的獲利來源，或者是在這種資訊不對稱能繼續運作的機制下，是不會想去改變的。」所以這也就不難理解為什麼始終沒有「評論」、「留言」的功能出現，因為這樣子資訊揭露就變多了，可能會影響跟店家收的 $$，或者是捅出沒有必要的麻煩來。（看看前幾天 catchplay 的事件就知道，有些資方真的是經不起說實話）我們在買網拍的時候，除了店家給的資訊之外，通常都能夠看到對賣家的評論、消費者的評論，而在人力銀行上時的資訊卻都是單向的，求職者間的訊息要到其他地方才能看得到。那為什麼求職的時候卻只能看單方給的資訊呢？雖然現在也有 ibeejobs，以及 ptt 上的 soft_job 、 tech_job 版，但其實最快的方法是能直接在職缺下方看到對這個工作的評價，讓壞的職缺或差勁的公司更快被淘汰，也讓好的職缺能夠更快找到想要的人才，假如 104、1111 他們有包袱無法去做出這些功能，那與其繼續抱怨或是癡癡等待，不如手動幫他們把這個功能給補上去，3. 身為一個軟體工作者能為這個就業環境做些什麼要改變台灣勞方的處境其實有一些難處，（看看 104、1111 就算對勞方不友善卻都仍然獨霸大部分人找工作的管道就知道）但我覺得可以試試看，而且能從我們軟體工作者開始做起。看到這裡應該會很多人覺得年輕人太天真了吧 XD第一個難處就是社會的風氣，身邊蠻多人不喜歡談論跟職涯相關的事情，但在 soft_job 版觀察的這段時間，我發現軟體工作者是相對樂於分享的，所以打算將 beta 版放在這裡讓大家試用看看，當有第一批人在上面分享以後，就像雞生蛋、蛋生雞了。我知道這個版上有許多人不認同 open source 的理念，不過就學寫軟體這件事來說，網路上分享的資源真的是比其他領域多上很多 ：）再來我也加上了匿名機制，不是為了亂講話，而是為了減少發表心得者的包袱。（馬的 說爛公司壞話就沒有公司敢 hire ，真的不知道是什麼玻璃心邏輯）第二個難處則是 使用者的習慣。就像前面說的，儘管人力銀行再怎麼不濟，大部分人要找工作都還是先把人力銀行的履歷打開。（我知道高階職缺會有 hunter 自己來找你 XD）所以額外再開個網站就會有跳出率的問題，最好的方式真的是讓使用者直接在原本的職缺下面就能夠留言討論，而 Chrome 在 desktop 的市佔率現在也到了很誇張的地步，（大家不信真的可以裝個 ga 看看 QQ，但我手機都用 CloudMosa 出品的 puffin 瀏覽器，Taiwan No.1）所以我想這是一個可以利用的槓桿，只要有 Chrome 插件，基本上就能讓大部分使用者有辦法用到這個功能了。所以身為一個軟體工作者，除了我們可以藉由分享、評論帶起這樣的風氣之外，這個 Chrome 插件的原始碼都是開放的，如果你認為有什麼功能要加上去，也隨時可以 fork 然後送 PR，或是開 Issue 給我，軟體的開發並不限於寫 code 而已，我想應該有蠻多其他專長能夠讓這件事更完善。因為我早上是個普通的上班族，所以可能就是下班後利用時間繼續開發。還是要聲明一下，我並沒有要從這個 Project 上面獲得什麼利益，就只是一個普通勞方，希望未來台灣的求職環境能更好一點。所以如果有人能夠幫忙分享、推廣，或是一起加入做這個 Project 的話，就太感激了 m(_ _)m雖然沒有人問到，不過這個插件名稱叫做「Clairvoyance - 求職天眼通」的原因蠻簡單的，因為我很喜歡看周星馳的電影：http://i.imgur.com/ueTxNy3.jpg<img alt="" src="//i.imgur.com/ueTxNy3.jpg"/>然後在 google 翻譯上輸入天眼通，就跳出來 Clairvoyance 這個結果 XD 覺得看起來也蠻屌的，而且也期許勞方們能一起練成天眼通，把勞資雙方的資訊不對稱看得透徹，讓爛職缺漸漸消失在勞力市場上，雖然接近要上班的凌晨，覺得自己把端午連假花在這個也不知道有沒有人會用的插件上，有一點蠢。不過 jserv 也說了嘛！有時候 open source
